Create Suspect

curl -d 'name=first media&phone_area=031&phone=123456&address=jl mayjen sungkana 88&city=soerabaia&end_of_contract=2021-11-30&business_field=internet' -X POST http://localhost:20215/insertsuspect

Prospect

Get Prospects

Syntax

curl localhost:20215/getlistprospectsbyname/< client_name>

Contoh

curl localhost:20215/getlistprospectsbyname/hanam

Create Prospect

Prospect dibuat dengan mengubah status Calon pelanggan menjadi P, dan ini dapat dilakukan dengan edit Client

Syntax

curl -d 'id=< client_id> &prospectdate=< YYYY-MM-DD > '&status=P -X POST http://localhost:20215/updateclient

Contoh

curl -d 'id=10320&prospectdate=2021-12-31'&status=P -X POST http://localhost:20215/updateclient
