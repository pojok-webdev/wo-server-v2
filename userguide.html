Suspect

Suspect atau Lead merupakan stage awal Calon Pelanggan

Create Suspect

Syntax
curl -d 'name= client_name &phone_area= phone_area &phone=& lt client_phone &address= address &city= city &end_of_contract= end_of_contract &business_field= business_field ' -X POST http://localhost:20215/insertsuspect
Contoh
curl -d 'name=first media&phone_area=031&phone=123456&address=jl mayjen sungkana 88&city=soerabaia&end_of_contract=2021-11-30&business_field=internet' -X POST http://localhost:20215/insertsuspect

Prospect

Get Prospects

Syntax
curl localhost:20215/getlistprospectsbyname/ client_name
Contoh
curl localhost:20215/getlistprospectsbyname/hanam

Create Prospect

Prospect dibuat dengan mengubah status Calon pelanggan menjadi P, dan ini dapat dilakukan dengan edit Client

Syntax
curl -d 'id= client_id &prospectdate= YYYY-MM-DD '&status=P -X POST http://localhost:20215/updateclient
Contoh
curl -d 'id=10320&prospectdate=2021-12-31'&status=P -X POST http://localhost:20215/updateclient

Survey

Get Surveys

Syntax
curl -d 'segment= baris_awal &offset= jumlah_baris ' -X POST localhost:20215/listsurveyproposal/
Contoh
curl -d 'segment=0&offset=10' -X POST localhost:20215/listsurveyproposal/

Pengajuan Survey

Syntax
curl -d "client_id= client_id &branch_id= branch_id &survey_date= survey_date &address= client_address &city= client_city &pic_name= client_pic &pic_phone= pic_phone &pic_email= pic_email " -X POST localhost:20215/proposesurvey
Contoh
curl -d "client_id=111&branch_id=1&survey_date=2021-11-30&address=Jl Gadjah Mada Sidoarjo&city=Sidoarjo&pic_name=Budi&pic_phone=0888123456&pic_email=budi@company.com" -X POST localhost:20215/proposesurvey

Update Pengajuan Survey

Syntax
curl -d 'id=survey_id&city=city_name&address=alamat_survey' -X POST localhost:20215/surveyupdate
Contoh
curl -d 'id=1897&city=Surabaya&address=Jl. Ciliwung No. 74-B' -X POST localhost:20215/surveyupdate

Install

Pengajuan Instalasi

Syntax
curl -d "client_id=client_id&client_site_id=client_site_id&install_date=install_date&address=installation_address&city=installation_city&pic_name=installation_pic&pic_phone=pic_phone&pic_email=pic_email" -X POST localhost:20215/proposeinstall
Contoh
curl -d "client_id=111&client_site_id=1&install_date=2021-11-30&address=Jl Gadjah Mada Sidoarjo&city=Sidoarjo&pic_name=John&pic_phone=123&pic_email=john@satya.com" -X POST localhost:20215/proposeinstall

curl -d 'install_request_id=1908&city=Surabayo&address=Jl. Ciliwung No. 74-B' -X POST localhost:20215/createinstallsite
